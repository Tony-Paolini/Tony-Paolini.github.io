Cheat Sheets

Below are three buttons to bring up a brief summary of basic HTML, CSS, and Javascript concepts!

HTML Cheatsheet

Category Concept Description
Basic Concepts Element An individual component in an HTML document, consisting of a start tag, content, and an end tag.
Attribute Additional information about an element, specified using name-value pairs within the start tag.
Tag A keyword or identifier surrounded by angle brackets, used to define elements in HTML.
Document Type Declaration (DOCTYPE) Specifies the version of HTML used in a document.
HTML Tags <!DOCTYPE html> Defines the document type
<html> Defines the root of an HTML document
<head> Contains metadata/information for the document
<title> Defines the title of the document
<body> Defines the body of the document
HTML Attributes class Defines one or more class names for an element
id Defines a unique id for an element
src Defines the URL of the resource to be used
href Defines the URL of the hyperlink

CSS Cheatsheet

Category Property Description
Text Properties color Specifies the text color.
font-family Specifies the font family for text.
font-size Specifies the font size for text.
font-weight Specifies the font weight for text.
Box Properties width Specifies the width of an element.
height Specifies the height of an element.
margin Specifies the margin space around an element.
padding Specifies the padding space inside an element.
Flexbox display Set to 'flex' to enable a flex context for all its direct children.
justify-content Aligns children within the main axis (e.g., center, space-between).
align-items Aligns children within the cross axis (e.g., center, flex-start).
flex-direction Defines the direction of the main axis (row, column).
Grid display Set to 'grid' to create a grid formatting context for its contents.
grid-template-columns Defines the size of the columns in the grid layout.
grid-template-rows Defines the size of the rows in the grid layout.
grid-gap Specifies the gap between rows and columns in a grid layout.
Pseudo-Classes :hover Applies a style when the user hovers over an element.
:focus Applies a style when the element has focus.
:active Applies a style when the element is being activated by the user.

JavaScript Cheatsheet

Basic Syntax Variables (var, let, const) Use var for function scope, let for block scope, and const for block scope constants.
Data Types Includes String, Number, Boolean, Null, Undefined, Object, Array, Function.
Operators Includes arithmetic (+, -), logical (&&, ||), comparison (==, !==), etc.
Conditionals Control flow with if, else, switch.
Loops Iterate with for, while, do-while.
Functions Declaration Declare with function keyword.
Expression Functions assigned to variables.
Arrow Functions Shorter syntax with =>, does not bind its own this.
DOM Manipulation querySelector Select elements with CSS-like selectors.
addEventListener Attach event handlers to elements.
innerHTML/innerText Manipulate element content.
Asynchronous JavaScript Promises Handle asynchronous operations.
async/await Simplify promise handling with async functions and await keyword.
Fetch API Perform network requests and handle responses.
